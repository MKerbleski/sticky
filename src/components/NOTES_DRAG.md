The following can be dragged. 

    MULIT NOTE CHANGES
    note -> note

    SINGLE NOTE CHANGES 
    note 1 & 2 & 3 -> top_level
    note 1 & 2 & 3 -> trash
    note 1 & 2 & 3 -> read
    note 1 & 2 & 3 -> watch
    note 1 & 2 & 3 -> todo

    LINK 
    link -> note 1 & 2
    link -> trash
    link -> read
    link -> watch 
    link -> todo
